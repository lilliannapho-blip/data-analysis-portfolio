# Tableau Chart Types

This project was completed as a university's coursework assignment to practice chart creation using different chart types using the provided dataset of Superstore.

## Datasets

This project uses the Tableau Sample Superstore dataset, which contains information on customer orders, products, sales, shipping, and profits.

## The Superstore Variables

The  dataset includes information such as:
- Order ID
- Ship Date	
- Ship Mode	
- Customer ID	
- Customer Name	
- Segment	
- Country/Region	
- City	
- State/Province	
- Postal Code	
- Region	
- Product ID	
- Category	
- Sub-Category	
- Product Name	
- Sales	
- Quantity	
- Discount	
- Profit

## Visualizations created
The chart types that were created using difefrent dimensions and metrics include:
- Box plot of discounts by region
- Bullet graph of tourism outbound by region
- Density chart of average life expectancy of a female and the average infant mortality rate
- Highlight table by region based on subcategories of the product
- Histogram of quantity purchased by segment
- Bubble chart of category profit by region
- Scatter plot of sales and profit
- Tree map of profit and sales by sub-category
- Combined chart of sales and profit by month
