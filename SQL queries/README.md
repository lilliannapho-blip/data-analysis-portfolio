# SQL queries

This project contains a collection of SQL queries developed as part of a SQL programming course. The queries were designed to extract, transform, and analyze data from the AdventureWorks database, focusing on customer and reseller information related to bicycle products.

The goal of this project was to practice database querying techniques and demonstrate the ability to retrieve meaningful insights from relational databases.

# SQL concepts demonstrated

The queries include the use of:

-INNER JOIN – Combining related records between tables where matching values exist
-LEFT JOIN – Retrieving all records from one table while including matching records from another table
-GROUP BY – Aggregating data to summarize information by categories

Aggregate functions:
-COUNT()
-SUM()
-AVG()
-MAX()
-MIN()
-WHERE clauses – Filtering data based on specific conditions
-ORDER BY – Sorting query results
-Subqueries – Using nested queries to perform more complex analysis

# AdventureWorks Database

Queries analyze data involving:
-Customers
-Resellers
-Product Categories
-Products
-Amount in Sales
-Date purchased
