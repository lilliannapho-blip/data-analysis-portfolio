# VLOOKUP function practice

This project demonstrates the use of Excel's **VLOOKUP** function to combine and analyze data from multiple sources. It was also completed as a university's coursework assignment to practice data integration in excel. 

The workbook contains two datasets:
- Website traffic data
- Keyword search data

Using VLOOKUP, the datasets were merged to match related records across worksheets. Additional Excel formulas were then used to calculate key performance metrics, including:
- Bounce rate
- Page views per visit
- Minutes per visit
- Click through rate
- Cost per click
- Cost per visit

## Skills Demonstrated
- Microsoft Excel
- VLOOKUP
- Data merging and lookup functions
- Basic KPI calculations
- Spreadsheet analysis
- Numerical formatting (% vs. $)
