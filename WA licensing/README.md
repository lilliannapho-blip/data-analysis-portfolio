# WA-licensing-program

This project is an interactive Python application that assists users in determining Washington State licensing information based on their responses to a series of prompts. The program guides users through a decision-making process and returns relevant licensing requirements or recommendations.

The project demonstrates how conditional logic and user input can be combined to create a simple decision support system.

Features:
-Interactive command-line interface
-Collects user input through prompts
-Displays licensing information based on user responses
-Handles multiple user scenarios

This project was developed as part of a Python programming course to practice user interaction, decision-making with conditional statements, and modular program design.
